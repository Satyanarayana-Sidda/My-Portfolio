# 🌌 DevOps & Cloud Interactive 3D Developer Portfolio

An ultra-premium, high-performance, and responsive WebGL-powered 3D landing page showcasing professional credentials, interactive cloud architectures, and milestones. Built from scratch using semantic HTML5, vanilla CSS3 glassmorphism systems, and custom Three.js WebGL animation physics.

---

* **Local Development Server**: `http://localhost:8080`

## 🎨 Signature 3D & WebGL Visual Features

This portfolio is packed with interactive 3D elements and smooth, hardware-accelerated visual matrices:

1. **🌀 Interactive 3D Holo-Core Skill Constellation**:
   A drag-and-touch active WebGL sphere connecting key skills to a glowing singularity core. Hovering over nodes scales them in 3D and streams real-time proficiencies and diagnostics to the console log.
2. **🔒 Holographic Verified Credentials Wall**:
   Futuristic digital security ID-style credential cards showcasing Oracle and IBM badges. Features:
   * **3D Mouse Tilts**: Cards tilt realistically according to the cursor's location.
   * **Iridescent Glare**: Shifting rainbow foil reflections that follow the mouse coordinates.
   * **Laser Scanner Sweeps**: Horizontal neon scanning sweeps that trigger on hover.
   * **LED Blinking Beacon**: Pulsing neon status LEDs indicating active validation.
   * **Cyber Brackets & Dot Matrix Grid**: Glowing structural corners and repeating grid patterns.
3. **🗺️ Interactive 3D Cloud Architecture HUD**:
   A live cloud network diagram (CDN, ALB, EC2, Lambda, RDS). Hovering over any node triggers glowing packet transfers and prints technical vector coordinates and diagnostic feeds directly to the HUD display.
4. **📸 3D Holographic Bio-Scanner Avatar Plate**:
   A circular profile plate that sways in a subtle 3D pattern, sways on mouse hover, features a sliding bioscanner laser scan line, and flips to showcase a custom cyber-grid backing with glowing credentials initials.
5. **🛣️ Scroll-Bound 3D Milestone Flips**:
   Career milestones in the timeline swing open dynamically on their Y-axis from left-to-center or right-to-center as you scroll, creating a premium perspective entry.
6. **🌊 Undulating Cyber-Terrain & Telemetry Bubbles (Footer)**:
   A gorgeous $32 \times 32$ vertex plane undulating smoothly using double-sine wave math. Features forty independent neon telemetry bubbles floating upward to simulate data streams.

## 🛠️ Technical Architecture

* **Frontend Structure**: Semantic HTML5 with custom responsive grids.
* **Styling System**: Raw CSS3 utilizing cohesive HSL color tokens, dark-mode themes, cybernetic glowing filters, and hardware-accelerated animations.
* **Graphics Engine**: Vanilla JavaScript running **Three.js** perspective camera rigs, custom buffer geometries, interactive raycasting tooltips, and performance-tuned background occlusion cullers.
* **Visual Assets**: Optimized custom framed illustrations and compressed JPEG textures.

## 🚀 How to Run Locally

To spin up a local development server and run the project locally:

### Option A: Using Python (Recommended)
Open a terminal in the project directory (`My-Portfolio`) and execute:
```powershell
python -m http.server 8080
```
Then, navigate to **`http://localhost:8080`** in your default web browser.

### Option B: Using Node.js
If you have Node.js installed, you can launch a quick static server instantly:
```powershell
npx http-server -p 8080
```
